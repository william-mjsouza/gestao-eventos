# Sistema de Gestão de Eventos

Sistema acadêmico para gerenciamento de eventos, inscrições, pagamentos e lista de espera.

## Tecnologias utilizadas

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven

## Funcionalidades

- Cadastro de eventos
- Cadastro de participantes
- Inscrição em eventos
- Controle de lotes
- Pagamentos
- Lista de espera

## Executar o projeto

```bash
mvn spring-boot:run
